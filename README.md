# AudioEnhancement-FullStackProject

Objective: Ensure the audio gets cleaned perfectly without making the voices sound robotic.

Task 1: Set up and configure the DeepFilterNet environment (handling PyTorch dependencies).

Task 2: Write a Python function that accepts a noisy .wav file, runs it through the deep learning model, and outputs a de-noised, de-reverbed .wav file.

Task 3: Tune the enhancement parameters (if necessary) to balance aggressive noise cancellation against vocal clarity.
